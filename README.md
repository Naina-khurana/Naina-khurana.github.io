### AI Regulatory Intelligence Pipeline

Built a FastAPI + PostgreSQL system to monitor government policy updates using RSS ingestion and document diffing. Implemented NLP classification using Hugging Face embeddings and translated regulatory text into structured JSON for decision-support dashboards.

### EEG Prediction of Psychiatric Disorders

Developed logistic regression and random forest models on EEG data from 945+ participants to identify patterns associated with psychiatric conditions. Evaluated performance and feature importance for clinical relevance.

### Full-Stack Booking Platform

Built a React + Node.js application with REST APIs and role-based dashboards supporting 100+ users and 50+ staff. Centralized scheduling, order tracking, and operational workflows.
